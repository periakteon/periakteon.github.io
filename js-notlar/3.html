Bir String'in Uzunuluğunu Bulmak (.length)

Bir string değerinin uzunluğunu, string değişkeninden veya string sabit değerinden sonra .length yazarak bulabiliriz.

console.log("Masum Gökyüz".length);

>>> 12

Boşluklar da uzunluğa dahil edilir. Bundan dolayı bize 12 değerini döner.

let lastNameLength = 0;,

const lastName = "Gökyüz";

lastNameLength = lastName.length;

console.log(lastNameLength);

>>> 7

Bracket Notation Kullanarak Bir String'in İlk Karakterini Bulmak

Köşeli parantez gösterimi (bracket notation), bir string içerisindeki belirli bir index/indisteki bir karakteri elde etmenin bir yoludur.

JavaScript gibi çoğu modern programlama dili, insanlar gibi 1'den saymaya başlamaz. 0'dan başlarlar. Buna Zero-based indexing denir.

Örneğin, Masum kelimesinin 0. indisindeki karakter M karakteridir. Yani const firstName = "Masum" ise, firstName[0] kullanarak dizgenin ilk harfinin değerini alabiliriz.

const firstName = "Charles";

const firstLetter = firstName[0];

console.log(firstLetter);

>>> C

Başka bir örnek için bkz.:

let firstLetterOfLastName = "";

const lastName = "Lovelace";

firstLetterOfLastName = lastName[0];

console.log(firstLetterOfLastName);

>>> L

String'in Değişmezliği (Immutability)

JavaScript'te, string değerleri değişmezdir, yani oluşturulduktan sonra değiştirilemezler.

Örneğin, Bob dizisindeki B harfi J harfine değiştirilemeyeceği için aşağıdaki kod bir hata üretecektir:

let myStr = "Bob";

myStr[0] = "J";

Bunun, myStr'ye yeniden bir değer atanamayacağı anlamına gelmediğini unutmayın. myStr'yi değiştirmenin tek yolu, ona şu şekilde yeni bir değer atamak olacaktır:

let myStr = "Bob";

myStr = "Job";

Bracket Notation Kullanarak Bir String'teki N. (Nth) Karakteri Bulma

Birinci karakteri bulabileceğimiz gibi n. (nth) karakteri de bracket notation kullanarak bulabiliriz.

Bilgisayarların saymaya 0'dan başladığını unutmayın, yani ilk karakter aslında sıfırıncı karakterdir.

const firstName = "Esra";

const secondLetterOfFirstName = firstName[1];

console.log(secondLetterOfFirstName);

>>> s

Bracket Notation Kullanarak Bir String'in Son Karakterini Bulma

Bir string'in son harfini elde etmek için string'in uzunluğundan (length) 1 çıkarmamız gerekir.

Örneğin, const firstName = "Esra" ise, firstName[firstName.length - 1] kullanarak string'in son harfinin değerini alabiliriz.

const lastName = "Ekici";

const lastLetterOfLastName = lastName [lastLetterOfLastName.length - 1];

console.log(lastLetterOfLastName);

>>> i

Bracket Notation Kullanarak Bir String'in Sondan N. (Nth-to-Last) Karakterini Bulma

Bir string'teki son karakteri almak için az önce kullandığımız aynı ilkeyi, sondan N.'ye kadar olan karakterleri almak için de kullanabiliriz. Örneğin:

const lastName = "Ekici";

const fourthToLastLetter = lastName[lastName.length - 4];

console.log(fourthToLastLetter);

>>> k
