Yorum Satırı Nasıl Eklenir?

JavaScript'te yorum satırı eklemenin iki yolu vardır:

I: // kullanarak veya,
II: /* */ kullanarak.

// olarak kullanılan yorum satırına in-line comment denirken,
/* */ olarak kullanılan yorum satırına multi-line comment denir.

// bu bir yorum satırıdır

/*
bu bir yorum satırıdır
*/
